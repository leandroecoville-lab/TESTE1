# LEGACY_NOTES — Notas operacionais (não executáveis)

Este repositório pode conter referências a instruções antigas de “modo avançado”.
A interpretação operacional suportada pela Fábrica é:

- **Execução profunda** = rodar `pytest`, rodar simulações (ex.: `meetcore-sim`) e validar gates (RUN_REPORT/APPROVAL + merge promoted).
- Qualquer instrução que conflite com governança, segurança ou contratos-first é tratada como **referência histórica** e não vira execução automática.

Objetivo: manter rastreabilidade sem permitir drift de políticas.
