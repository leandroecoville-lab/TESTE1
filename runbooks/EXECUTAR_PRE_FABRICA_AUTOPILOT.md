# Runbook — Autopilot Pré-Fábrica (V011-2.0)

## Como usar no GPT Builder (na prática)
1) Cole sua ideia / pesquisa
2) Comece com:
- `/SCAN` para resposta rápida, ou
- `/DILIGENCE` para análise completa

## Inputs recomendados (formulário)
TEMA:
OBJETIVO/DECISÃO:
PAÍS/MERCADO:
RECORTE:
NÍVEL: /SCAN ou /DILIGENCE
FORMATO FINAL:
CONSTRAINTS:
FONTES PRIORITÁRIAS:
PRAZO/URGÊNCIA:

## Saída obrigatória do GPT
- VS4
- 3 rodadas (R1/R2/R3)
- Evidências + Log de crenças
- Red Team + Patch
- Handoff Pack (arquivos/schemas)
